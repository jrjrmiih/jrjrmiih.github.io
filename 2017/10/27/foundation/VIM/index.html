常用命令

  • 编辑：

    " 删除行尾空格
:%s/\s\+$//

" 将tab替换成空格
:%s/\t/    /g

  • 查看：

    " 查看被跟踪的文件
$ git ls-files

" 查看被忽略的文件
$ git ls-files -o  // -o, --others

" 查看文件为什么被忽略
$ git check-ignore -v <file/folder name>  // -v, --verbose

自定义的快捷键

  • 默认快捷键：

    gt: 跳转到下一个tab窗口
gT: 跳转到上一个tab窗口
<c-w>z/<c-w><c-z>/:pc: 关闭预览窗口
<c-g>: 显示当前文件路径

  • 自定义快捷键:

    leader: <SPACE>
<leader>ss: 开启关闭拼写检查

Vundle 插件使用

  • 安装：

    $ git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

  • 安装配置好的插件：

    :PluginInstall

    或者

    $ vim +PluginInstall +qall

Denite 插件使用

  • 安装：

    Plugin 'Shougo/denite.nvim'
Plugin 'Shougo/neomru.vim'  // 检索最近打开文件的插件

  • 配置：

    " 使用rg作为grep内容检索命令
call denite#custom#var('grep', 'command', ['rg'])
call denite#custom#var('grep', 'default_opts', ['--vimgrep', '--no-heading'])
call denite#custom#var('grep', 'recursive_opts', [])
call denite#custom#var('grep', 'pattern_opt', ['--regexp'])
call denite#custom#var('grep', 'separator', ['--'])
call denite#custom#var('grep', 'final_opts', [])

" 使用rg作为file_rce文件递归检索命令
call denite#custom#var('file_rec', 'command', ['rg', '--files', '--glob', '!.git', ''])

" 使用连续子字符串，作为file_rec检索条件
call denite#custom#source('file_rec', 'matchers', ['matcher_substring'])

  • 快捷键：

    <leader>uf: 检索当前目录
<leader>ur: 递归检索当前目录及子目录文件
<leader>um: 查看最近使用文件
<leader>ub: 查看已打开文件
<leader>ul: 检索当前文件内容
<leader>ug: 递归检索当前目录及子目录文件内容

YouCompleteMe 插件使用

  • 安装：

    Plugin 'Valloric/YouCompleteMe'
$ cd ~/.vim/bundle/YouCompleteMe
$ python3 ./install.py --clang-completer  // 使用python3解释器，可以补全python3

  • 配置：

    let g:ycm_confirm_extra_conf = 0  // 当打开支持类型文件时，自动确认使用该插件

AutoFormat 插件使用

  • 安装：

    Plugin 'Chiel92/vim-autoformat.git'
$ sudo pip install yapf  // python 格式化支持

  • Markdown 格式化支持：

    $ npm install -g remark-cli

  • 快捷键：

    <leader>l: 格式化代码

TagList 插件使用

  • 快捷键：

    <leader>t: 标记列表

NerdCommenter 插件使用

  • 安装：

    Plugin 'scrooloose/nerdcommenter'

  • 配置：

    let g:NERDSpaceDelims = 1  // 在注释符与注释之间加入空格
let g:NERDTrimTrailingWhitespace = 1  // 取消注释时，删除多余空格
let g:NERDCommentEmptyLines = 1  // 允许注释空行，在注释大段代码时有意义

  • 快捷键：

    <leader>/: 注释与反注释代码

Ultisnips 插件使用

  • 安装：

    Plugin 'SirVer/ultisnips'
Plugin 'honza/vim-snippets'  // 各语言补全模板插件

  • Markdown 自定义：

    ###
-

  • 快捷键：

    <c-h>: 扩展代码片段
<c-j>: 跳到下个标记处
<c-k>: 跳到上个标记处

LiveDown 插件使用

  • 安装：

    $ npm install -g livedown
Plugin 'shime/vim-livedown'

  • 快捷键：

    <leader>gm: 打开默认浏览器，实时预览Markdown文件

备注：当发现LivedownToggle不生效时，先执行一次LivedownKill。
